# Home Server

This is the documentation for my home server — a self-hosted Proxmox setup running media automation, home automation, monitoring, and various other services. Everything here is written from what I actually run, not from theory.

The server runs on a single Proxmox VE node. Services are split between LXC containers (for things that need direct hardware access, like Plex) and Docker containers inside VMs (for everything else).

## What's documented here

| Section | What it covers |
|---------|---------------|
| [Media Automation](/docs/home-server/media-automation/overview/what-this-stack-does) | Plex + Usenet stack: Sonarr, Radarr, Prowlarr, SABnzbd, Seerr |
| Home Automation | Home Assistant setup, automations, and integrations |

## Infrastructure at a glance

- **Host**: Proxmox VE 8.x, bare metal
- **Storage**: ZFS mirror pool (`tank`) for media, NVMe for system disks
- **Networking**: Flat LAN, Cloudflare Tunnel for external access where needed
- **Monitoring**: Prometheus + Grafana + Telegram alerts
