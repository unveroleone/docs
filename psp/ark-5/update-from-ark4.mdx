# Update from ARK-4

If you are already on ARK-4, updating to ARK-5 does not require a full reinstall. You download the update file, run it on the PSP, and it handles the rest.

PS Vita users must update to **Adrenaline-8** before proceeding.

## Steps

**1. Connect your PSP to a computer**

Connect your PSP via USB or remove the memory card and use a card reader.

**2. Copy the update file to the memory stick**

Download the **ARK-5 UPDATE** file from the [FasterARK releases page](https://github.com/PSP-Arkfive/FasterARK/releases/tag/latest). Copy the `UPDATE` folder to `/PSP/GAME/` on your memory stick.

**3. Launch ARK Update**

Safely disconnect the PSP or memory card from the computer. On your PSP, go to the **Game** tab in the XMB. You should see **ARK Update** listed there. Launch it and wait for it to complete. The PSP will restart back to the XMB automatically.

The updater may say you are downgrading. This is expected — the version numbers were changed during development. It is not a problem.

**4. Clean up and verify**

Once it completes, press **Square** to delete the updater. Then press **Select** and confirm the CFW shows ARK-5.
