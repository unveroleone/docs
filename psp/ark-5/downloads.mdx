# Downloads

The latest stable release is on the [FasterARK GitHub Releases page](https://github.com/PSP-Arkfive/FasterARK/releases/tag/latest).

There are two packages: **FasterARK Lite** (`FasterARK_psp.zip`) and **FasterARK Full** (`FasterARK_psp_full.zip`). Both let you install and run ARK-5. The difference is the additional tools bundled in Full.

## Comparison

| | FasterARK Lite | FasterARK Full |
|---|---|---|
| FasterARK launcher | Yes | Yes |
| CustomIPL | Yes | Yes |
| DC10 recovery tool | No | Yes |
| Overclock Stress Tester | No | Yes |
| Custom Launcher | No | Yes |
| VSH Menu | No | Yes |
| Translations | No | Yes |
| LEDA Plugin | No | Yes |
| ARK-150 Addon | No | Yes |

If you only want to install ARK-5 and make it permanent, **Lite** is enough.

## What is included in each

### FasterARK Lite

- **FasterARK** — the launcher used to boot into CFW
- **CustomIPL** — flashes ARK-5 to make it persist after power off

### FasterARK Full

Everything in Lite, plus:

- **DC10** — a recovery tool for the PSP. See the [DC10 GitHub](https://github.com/PSP-Arkfive/Despertar-Del-Cementerio) for installation and usage.
- **Overclock Stress Tester** — an app to find the maximum stable overclock your specific PSP supports
- **Custom Launcher** — an open source homebrew replacement for the XMB
- **VSH Menu** — a hub for the XMB
- **Translations** — language files for both the XMB and Custom Launcher
- **LEDA Plugin** — lets you run PSP 1.50 homebrew on any firmware and any PSP or Vita model
- **ARK-150 Addon** — for PSP 1000 only. Installs the full 1.50 ARK firmware to the memory stick so you can dual-boot alongside the latest firmware. See the [ARK-150 GitHub](https://github.com/PSP-Arkfive/ARK-150) for installation and usage.

## Also available

- **ARK_UPDATE.zip** — for ARK-4 users whose OTA updates are not working. Download, copy the `PSP` folder to your memory stick, and run the Updater.
