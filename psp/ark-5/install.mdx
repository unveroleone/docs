# First Time Install

This installs ARK-5 as a **temporary custom firmware**. Temporary means the CFW runs until you power off your PSP. After a full power off, you need to launch FasterARK again to restore CFW functionality.

This is useful if you want to try ARK-5 before committing. Once you are ready to make it permanent, follow the [Making it permanent](/docs/psp/ark-5/permanent) guide.

Before starting, make sure you have the right package. See [Downloads](/docs/psp/ark-5/downloads) if you have not downloaded anything yet.

## Steps

**1. Connect your PSP to a computer**

Connect your PSP via USB or remove the memory card and use a card reader.

**2. Copy FasterARK to the memory stick**

Download and extract `FasterARK_psp.zip` (or `FasterARK_psp_full.zip` for the Full variant). Extract the package directly onto the root of your memory stick. If a `PSP` folder already exists on the memory stick, merge it — do not replace it entirely.

**3. Launch FasterARK**

Safely disconnect the PSP or memory card from the computer. On your PSP, go to the **Game** tab in the XMB. You should see FasterARK listed there. Launch it and wait for it to complete. The PSP will restart back to the XMB automatically.

**4. Verify CFW is active**

Press **Select** on your PSP. A menu should appear with text in the top right corner of the screen. If you see it, ARK-5 is running.

To see what features are available, visit the [ARK-5 GitHub](https://github.com/PSP-Arkfive/ARK-5).

## Next step

To make ARK-5 permanent so it survives a power off, continue to [Making it permanent](/docs/psp/ark-5/permanent).
