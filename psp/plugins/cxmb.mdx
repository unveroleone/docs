# CXMB

CXMB allows you to install custom XMB themes made by the community. It works on the XMB only and has no effect on the Custom Launcher, which uses its own separate theme system.

**Runlevel:** VSH (XMB) only.

## Download

[**Download CXMB**](https://wololo.net/talk/download/file.php?id=3995&sid=e959a08b2f9096913455fac40e7ef84c).

## Themes

CXMB enables CTF themes, the community format with deeper customization than native PTF themes. For the full explanation and theme downloads, see [XMB Themes](/docs/psp/themes/themes).
