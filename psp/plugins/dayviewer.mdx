# DayViewer

DayViewer displays customizable information on the XMB: date, time, battery percentage, and more. A separate plugin called `dayviewer_game` extends this to in-game use.

## Variants

**DayViewer** — displays info on the XMB.

**Runlevel:** VSH (XMB) only.

**DayViewer for Game** — displays info while inside a game.

**Runlevel:** GAME only.
