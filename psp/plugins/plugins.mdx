# Plugins

ARK-5 supports third-party plugins (.prx files) that extend XMB and in-game functionality. The plugins listed in this section have been tested with ARK-5 and confirmed working.

## Installation

**1. Create the seplugins folder**

Create a `seplugins` folder at the root of your memory stick if it does not exist yet.

**2. Place the plugin**

Copy the plugin's `.prx` file into `/seplugins/`. Some plugins require a specific subfolder — check the individual plugin page for details.

**3. Open Plugin Manager**

On your PSP, go to the **Extra** tab and open **Plugin Manager**.

**4. Set the plugin location**

Set where the plugin will run: **VSH/XMB** (the PSP's main menu), **GAME**, or **Always**. Some plugins specify a required location (otherwise psp could malfunction)
