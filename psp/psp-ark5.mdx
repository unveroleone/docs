# ARK-5 CFW on PSP

ARK-5 is a custom firmware (CFW) for the PSP, currently in active development. It is the successor to ARK-4, which itself succeeded ARK-3, PRO, and LME.

For information on ARK-4, see [PSPunk's ARK-4 page](https://www.pspunk.com/psp-ark4/). For a first-hand account of installing ARK-5 on a PSP 1000, read the [blog post](/blog/psp-ark5).

## What ARK-5 adds

Compared to older CFW like PRO and LME, ARK-5 includes:

- Compiled with the latest PSPSDK
- Better plugin management in the XMB
- True overclocking for all PSP models (not available on PS Vita)
- Improved memory management for consoles with extra RAM
- Easier installation and usage

For the full details, see the [ARK-5 GitHub](https://github.com/PSP-Arkfive/ARK-5).

## Who this guide is for

This guide covers three starting points:

1. **No CFW installed** — you are on official firmware and want to install ARK-5 for the first time
2. **ARK-4 installed** — you want to update to ARK-5
3. **PRO or LME installed** — you want to move to ARK-5

If another guide refers you to PRO or LME, you can ignore that. ARK-5 is the only firmware currently being maintained and updated.

## Prerequisites

Before installing, make sure your PSP meets the following conditions:

- Running official firmware **6.61**. If you are on a lower version, follow the [firmware update guide](/docs/psp/prerequisites/update-firmware) first.
- If you currently have **permanent CFW** like PRO with Infinity 2 installed, you need to reinstall official firmware before proceeding. Follow the [Chronoswitch guide](/docs/psp/prerequisites/chronoswitch) to do that.
