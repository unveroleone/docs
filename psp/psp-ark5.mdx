# ARK-5 CFW on PSP 1000

A quick guide to installing ARK-5 beta custom firmware on an original PSP 1000 (Fat).

## What you need

- A PSP 1000 (any firmware version)
- A Memory Stick Duo (or microSD adapter)
- A USB cable to connect the PSP to your computer
- The ARK-5 release package from the [official GitHub](https://github.com/PSP-Archive/ARK-4)

## Why ARK-5

ARK-5 is a modern CFW that runs on all PSP models. The beta builds include fixes for the 1000 model that older CFWs never addressed. Cold-boot support, plugin management, and ISO loading all work out of the box.

## Installation overview

1. Format the Memory Stick on the PSP
2. Copy ARK-5 files to the stick root
3. Run the installer from the PSP game menu
4. Flash the CFW to internal storage
5. Reboot and hold the trigger button to enter recovery
