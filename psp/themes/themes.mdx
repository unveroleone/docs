# XMB Themes

The PSP supports two theme formats: **PTF** and **CTF**. They work differently and cannot be used at the same time.

## PTF themes

PTF is the native Sony theme format. No plugin is required. You install a `.ptf` file directly through the PSP's theme settings.

## CTF themes

CTF is a community format that allows much deeper customization than PTF. CTF themes require the [CXMB](/docs/psp/plugins/cxmb) plugin to work.

**Important:** if CXMB is enabled, PTF themes will not work. To go back to a PTF theme, disable CXMB in the Plugin Manager first.

## Theme sources

[**PSPunk Theme Library**](https://www.pspunk.com/psp-themes/) — a large collection of community-made themes, searchable by style and category.
